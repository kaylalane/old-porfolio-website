Idiot's Guide to HTML: The Basics

From your local English student who codes.


HyperText Markup Language (HTML) builds pages for websites, and this lesson teaches you how to make a simple webpage. An integrated development environment (IDE), or code editor, isn't required to begin learning HTML. If you don't have an IDE available use your computer's built-in text editor, Notepad on Windows and TextEdit on Mac, or download Visual Studio Code for free.

HTML Elements

An HTML page is written with HTML elements containing an opening tag, a closing tag and the information written between those tags. A tag opens with <tag_name> and closes with </tag_name>, the / is only for the closing bracket.

HTML Document Structure

Before personalizing your document, you need four basic elements:

  • <!DOCTYPE html> is the first line of every HTML document, doesn't need a closing tag and declares the document as an HTML document.
  • <html> nestles every other element within itself. <html> is the second line of every document and </html> is the last line of every document.
  • <head> will be important later but it carries metadata and links to CSS and Javascript files live here.
  • <body> carries the bulk of your webpage's information.

HTML Building Blocks

Notice the tag <p>? Content within the <p> tag shows up as paragraph text, like the words you're seeing now.

Let's add a couple of headings with the <h1> tag before the paragraph.

<h2> Typee <h2>
<h3> Benito Cereno <h3>
